引子：Agent 从云端坠落到你的桌面

过去两年，AI Agent 一直悬在云端。你在网页上跟它聊天，它在服务器上运行，数据要来回传，文件要手动上传下载。Agent 很聪明，但始终隔着一层"云"。

Kimi Work 要做的是把 Agent 搬到你电脑上。不是网页里那个会聊天的 Kimi，而是一个能读懂你的本地文件、操控你的浏览器、在你的电脑上自主干活的桌面助手。

6月3日，月之暗面（Moonshot AI）发布 Kimi Work 公测版，Mac 和 Windows 双平台同步上线。内核是已获十几万日活验证的 Kimi Code，底层驱动是开源模型 Kimi K2.6——支持最多 300 个子 Agent 并行协作、连续工作 13 小时、4000+ 次自主工具调用。

二、技术硬核：三大支柱撑起桌面Agent

2.1 Kimi K2.6：专为Agent编排设计

K2.6 是月之暗面 4月开源的旗舰模型，1T 参数 MoE 架构（32B 激活参数），256K 上下文。

能力 数据
不间断编码 13 小时连续，4000+ 行代码
工具调用 4000+ 次自主调用（浏览器/API/本地命令）
Agent 集群 300 个子 Agent 并行协作
多模态 原生支持图片+视频输入

关键区别：K2.6 在设计时就以 "Agent 编排" 为目标，不是事后拼凑的 tool-calling 方案。底层架构就是为多智能体设计的，所以能稳定支撑 300 个 Agent 同时跑。

2.2 Agent 集群：300 个 AI 同时为你干活

传统 Agent 是"想一步、做一步、看结果、再想一步"的串行流程。Kimi Work 打破了这种模式：

Kimi Work 自动拆解：

│   ├─ 同花顺抓取A股新能源板块
│   ├─ Yahoo Finance 抓美股数据

├─ Agent 21~25：报告撰写组

└─ Agent 26~30：质量审核组
    ├─ 数据校验 + 格式排版
    └─ PPT 生成

2.3 WebBridge：让 AI 真的会"上网"

大部分 AI 工具只能通过 API 访问网页，Kimi Work 的 WebBridge 可以像人类一样操控浏览器——点击、滚动、填表。据社区实测，Claude Code 的 Chrome 插件在处理公众号后台这类动态渲染页面时会直接报错，而 Kimi Work 可以正常处理。

安装完成后打开 Kimi 客户端，在左侧导航栏从 「Chat」 切换到 「Work」 标签页——界面立刻从"聊天框"变成"项目管理面板"。

Step 3：创建第一个任务

"帮我调研过去一周 AI 领域的重大新闻，整理成一份 10 页的市场简报 PPT。注意：重点关注国内大模型和 Agent 方向。"

Kimi Work 会自动：拆解任务 → 通过 WebBridge 访问新闻网站 → 筛选重点 → 整理分析 → 调用内置 Skill 生成 PPT → 保存到本地文件夹。

选谁？看你是什么人

我的真实体验：这篇 Kimi Work 评测文章就是 WorkBuddy 的活广告——从搜索热点、查阅官方文档、整理定价方案到生成公众号HTML，全程一个对话窗口搞定。你正在读的每一个字，都是 WorkBuddy 写的。

但公平地说，Kimi Work的 Agent集群 能力是 WorkBuddy 目前没有的。300个子Agent并行拆解复杂研报——这种规模的并行编排，WorkBuddy的专家团模式做不到同样级别的"暴力并行"。

结论：不是谁替代谁，而是谁更适合你的场景。 金融数据需求重选 Kimi Work，办公生态需求重选 WorkBuddy。两个都装——没冲突，各取所长。

七、优势与槽点

✅ 五大优势

1. 零门槛 GUI，不用写代码
2. Agent 集群独一份：300 子 Agent 并行
3. WebBridge 杀手锏：动态渲染页面也能抓
4. 专业数据生态：同花顺/天眼查原生接入
5. 本地安全：数据不出设备

⚠️ 四个槽点

1. Agent 集群门槛高：¥199/月起
2. Beta 阶段稳定性待验证
3. macOS 仅支持 M 系列芯片
4. 92% AI 生成代码，Bug 概率可能更高

结语：Agent 的"iPhone 时刻"

2026年3月，黄仁勋在 GTC 上说："未来计算机不运行应用程序，只运行 AI Agent。"

三个月后，Kimi Work 把它做出来了。你不再打开 Word 写报告、打开 Excel 做分析、打开浏览器查数据——你只打开 Kimi Work，跟它说一句话，它替你完成这一切。

当然，Kimi Work 还不是 iPhone。它更像是 iPhone 刚发布时——惊艳，但不完美。Bug 有，稳定性待打磨，Agent 集群要掏钱。

但方向是对的。AI 从"陪你聊天"到"替你干活"，桌面 Agent 是不可避免的一站。

要不要现在就上车？如果你是需要处理大量信息的金融/科研/分析从业者，值得——¥199/月换来每周省下20小时。如果你是偶尔用 AI 的轻度用户，先用免费版 Adagio 体验就好。

建议亲自下载试一下——免费版有 6 个 Agent 任务额度，足够体验核心能力。不点一下鼠标，永远不知道 AI 替你干活的快感。
