## 一句话总结

ArkClaw 是火山引擎（字节）把开源 OpenClaw「搬上云」的托管版——核心不是又能干啥，而是 **你不用管部署、不用管运维、关机任务照样跑**。**适合不想折腾本地环境、重度依赖飞书办公、希望 7×24 在线跑定时任务的人；但它是云端虚拟机，碰不到你电脑本地文件，且强绑定火山生态，建议把它当「云端的办公智能体」，而非本地数字员工。**

> **评测说明**：本文基于 ArkClaw 官方资料（火山引擎产品页、方舟 Coding Plan 文档、Skills Hub 说明、ArkClaw 使用教程 36986 / 36990）与多家媒体公开实测（PConline 2026-03 深度测评、极客公园 / 新浪财经 2026-03、Volcengine 官方文章 36891 / 36385、openclawai.net 英文评测）整理。站长赵生已实际体验，确认产品可正常运行；文中「实测反馈」均引自上述公开来源，具体功能与定价以火山引擎官方最新页面为准。

## 核心功能评测

### 1. 云端一键部署与免运维 ⭐⭐⭐⭐⭐（5/5）

ArkClaw 最被夸的就是「把运维扔给平台」。本地 OpenClaw 的痛点——Docker 拉不起来、Token 跑光进程死、半夜定时任务没跑——在云端基本不存在。官方文档（36986）写明：订阅 Coding Plan → 方舟体验中心左侧「Agent → ArkClaw」→ 点「立即开始」→ 约 1~2 分钟实例跑起来，无需配环境、无需写代码。

极客公园实测：「在火山方舟控制台找到 ArkClaw 入口，开通会员，点立即创建，等了两分钟，一个云端 OpenClaw 实例就跑起来了」（极客公园 / 新浪财经，2026-03）。openclawai.net 英文评测给的对比更直白：专属 ECS 资源，进程不因关机 / 网络波动中断，本地部署常见的「漏跑定时任务、对话上下文丢失」在这里基本不发生。

**差异化长板**：7×24 在线。对比 AutoClaw / Kimi Work / QoderWork 等本地方案「关机即离线」，ArkClaw 关电脑后任务仍在后台跑、记忆不丢——这是它和本地派最本质的区别。

### 2. 飞书原生深度集成 ⭐⭐⭐⭐⭐（5/5）

这是 ArkClaw 最被媒体称赞的能力。飞书接入最省心：打开 App 扫码就能用，配置完成后它能以你的身份直接操作文档、日历、任务、在群里回消息。官方飞书插件六大能力覆盖 90% 办公场景：消息（读群聊 / 发消息 / 搜记录）、文档（建读编云文档 / 回评论）、多维表格（增删改查 / 筛选 / 视图）、电子表格（Excel / 飞书表格读写）、日历（查档期 / 建会 / 协调时间）、任务（建任务 / 派负责人 / 同步待办）。

PConline 实测的一个场景很说明问题：前一晚同事在飞书群一句话约好第二天 9 点会议，「@龙虾」时间、参会人、日历提醒全自动搞定，全程没切出飞书窗口。另一位用户让它「总结今早会议纪要、提取 3 个行动项」，它直接读飞书文档输出结构化结果。

openclawai.net 实测补充：「让它总结今早会议纪要并提取三个行动项，它直接读飞书文档输出结构化结果，没有废话。对主要工作在飞书的人，这个功能实用价值很高。」

### 3. Skills Hub 与安全架构 ⭐⭐⭐⭐⭐（5/5）

ArkClaw 的安全叙事是其最大差异化之一。PConline 长文详述了本地养虾的「翻车现场」：CVE-2026-25253（CVSS 8.8，在野利用）网关 token 被盗、4.2 万个实例零认证暴露公网、ClawHub 技能市场 12%~20% 恶意包投毒。ArkClaw 用四层防护应对：

- **平台安全**：控制面默认不暴露公网、强制认证、独立隔离环境
- **供应链安全**：自研 Skills Hub 替掉野生 ClawHub，上架前安全扫描 + 代码审计，运行持续巡检
- **运行时安全**：事前意图识别拦截高危指令、事中删文件 / 群发等不可逆操作二次确认、事后不可篡改日志审计
- **身份权限**：凭证安全托管（Agent 只拿临时低权限令牌，碰不到原始密钥）、最小权限原则、一键撤销

实测中「删文件、群发消息这类不可逆操作必须二次确认，再不会出现叫 STOP 都叫不住的名场面」（PConline，2026-03）。对普通用户，等于把「防火防盗」交给银行级平台。

### 4. 多模型协同与 Hermes 自沉淀 ⭐⭐⭐⭐（4/5）

默认 Doubao-Seed-2.0 系列（Code / Pro / Lite），兼容 GLM / Kimi / DeepSeek / MiniMax2.5，支持「Auto」智能调度——按任务复杂度自动选模型。极客公园实测：「简单指令用轻量模型快速响应，复杂任务调强模型保质量，响应速度比之前方案快、成本也降了。」

新增的 **Hermes Agent** 是进阶亮点：自动把已解决的问题沉淀为可复用技能，越用越专业；ArkClaw 负责连接与决策，Hermes 专注执行与学习，调度 + 执行双引擎。官方还推「金融理财专家」「漫剧创作专家」等专属 Agent 团队（约 ¥40~59/月订阅），3 位 AI 专家 24 小时待命。

### 5. 成本与实测表现 ⭐⭐⭐⭐（4/5）

订阅制是 ArkClaw 对抗「Token 焦虑」的武器。极客公园点出关键：「Agent 类工具高频调用，一个复杂任务可能触发几十次模型交互。按量计费让你每发一条指令都在默算成本；订阅制让你放心用。」价格上，Coding Plan Lite ¥9.9 起（曾附赠 7 天 ArkClaw 体验，2026-03-24 起 Lite 新购不再送），Pro ¥49/月 订阅期内免 ArkClaw 全功能。

实测数据：少数派用户实测办公任务「完成准确率达 90% 以上」（Volcengine 文章 36891）；结合 Coding Plan 长期使用成本比按量计费「降低约 30%」（同文，基于少数派实测）。首购活动还送 5000 万 Token（六月限时），进一步降低尝鲜成本。

## 优势与短板

### 优势

**真正免运维。** 专属 ECS、自动升级、一键修复，用户只需打开对话窗。本地部署的环境配置、版本升级、安全补丁全由平台兜底。

**飞书深度打通。** 扫码绑定即以你身份操作文档 / 日历 / 任务，办公场景「在群里把事办了」，不用切窗口。这是字节生态的先天优势，其他厂商难复制。

**安全托底。** 四层防护把本地养虾的漏洞（公网暴露、技能投毒、误删邮件）系统性解决，普通用户不必当安全专家。

**订阅制省心。** Coding Plan 包月告别 Token 按量焦虑，高频调用也不肉疼。

### 短板

**碰不到本地。** 跑在云端虚拟机，不能像本地版读写你电脑文件、控制本地桌面浏览器、操作本地应用——自由度换来了安全与便捷。

**数据保留窗口短。** Coding Plan 到期后 ArkClaw 被删，数据仅留 24 小时，长期归档需自行导出。

**性能分层。** 轻量版算力有限，多并发推理要高级版（8C16G），成本随需求上升。

**生态绑定。** 深度依赖火山引擎（TOS 存储、方舟模型、飞书），跨云迁移成本高。

**活动有时效。** 5000 万 Token 赠送等首购优惠限六月，长期成本看套餐档位。

## 最终推荐

### 强烈推荐给

- **飞书重度办公用户**：原生集成，在群里直接把事办了，不用切窗口
- **不想折腾本地的用户**：零部署零运维，打开浏览器就能养虾
- **需要 7×24 定时任务的人**：关机后任务照跑、记忆不丢
- **怕本地安全风险的非技术用户**：四层平台防护替代自己当安全专家

### 谨慎考虑

- **需要处理本地文件 / 控制本地桌面的用户**：云端摸不到你电脑，选本地派（AutoClaw / QoderWork）
- **跨云 / 多云用户**：强绑定火山生态，迁移成本高
- **成本敏感且低频用户**：轻量版性能有限，按需评估套餐
- **要求数据长期本地留存**：到期后仅留 24 小时，需自行导出

### 一句话建议

**ArkClaw 不是「又一个云端龙虾」，而是把 OpenClaw 的部署、运维、安全全收拢到平台侧的托管版。** 它的差异化在「免运维 + 飞书深融 + 四层安全」，最适合重度飞书办公、要 7×24 在线、不想管本地环境的人；要碰本地文件或完全数据自主，回头看 AutoClaw / QoderWork。
