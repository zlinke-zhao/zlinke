### 重大升级 - LangChain 1.x + LangGraph（21:30）
- **触发原因**：赵生提问"既然 LangChain 已更新到 1.x，为何非要使用 0.2.x？"
- **背景**：LangChain 1.x 进行了大规模重构，Agent 功能迁移至独立的 `langgraph` 包
- **升级内容**：
  - 依赖：`langchain>=0.2.0,<0.3.0` → `langgraph` + `langchain-openai`（不再需要锁版本）
  - Import：`from langchain.agents import create_react_agent, AgentExecutor` → `from langgraph.prebuilt import create_react_agent`
  - 工具定义：`Tool(name=..., func=..., description=...)` → `@tool` 装饰器（更简洁，自动提取 docstring）
  - Agent 创建：`create_react_agent(llm, tools, prompt) + AgentExecutor(...)` → `create_react_agent(model=llm, tools=..., prompt=...)`（一行搞定）
  - 运行方式：`agent_executor.invoke({"input": ...})` → `agent.stream({"messages": ...}, stream_mode="values")`
  - 查看思考链：`verbose=True` → `pretty_print()` 配合 `stream_mode="values"`
  - 移除：手动 ReAct 提示模板（LangGraph 内部已封装）
- **优势**：
  - 代码更简洁（从 60+ 行减到 40+ 行）
  - 无需手写 ReAct 模板，降低学习门槛
  - 使用最新稳定版，读者 `pip install` 直接装最新版即可
  - `stream` 模式输出更结构化（HumanMessage / AIMessage / ToolMessage）
